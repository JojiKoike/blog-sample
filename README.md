# blog sample

[![Build Status](https://travis-ci.org/suzuken/blog-sample.svg?branch=master)](https://travis-ci.org/suzuken/blog-sample)

A kind of blog.

## Prerequisite

* Docker
* Go (>= 1.7)
* sqlite3

## Bootstrap

    make deps

## How to run

    make run
